## נרטיב פיץ' — מבנה 5 חלקים

### 1. צורך (Problem)
**הכותרת:** "220,000 זכאים — רק 66,000 מממשים"

ישראל מוציאה ₪20 מיליארד שנה על סיעוד. 70% מהזכאים לא מקבלים שירותי מניעה.
לא מחוסר כסף — מחוסר מידע, בירוקרטיה, חסמי שפה, ומחוסר ליווי אישי.
כל שנה שמאחרים = הידרדרות שעולה פי 3 לטפל בה מאוחר יותר.

### 2. פתרון (Solution)
**הכותרת:** "AI שמכיר כל אדם — ומסביר למה"

סל אישי = אלגוריתם התאמה 5-שכבות + Agentic AI + 5 ממשקים דיגיטליים.
102 שירותים ממופים × פרופיל אישי = 10,404 התאמות מחושבות לכל אזרח.
כל המלצה מוסברת: "הומלץ כי: ניידות מוגבלת + מוטיבציית שייכות + בשכונה שלך."
אינטייק קולי 5 דקות במקום טופס 8 עמודים.

### 3. ערך (Value)
**הכותרת:** "דחיית הידרדרות = חיסכון ₪2B שנה"

- מימוש זכויות: 30% → 75% (+45 נקודות)
- SDI: מ-0 ל-17+ שירותי מניעה פעילים לאדם
- דחיית הידרדרות: 20% = ~3 חודשים נוספים של עצמאות
- חיסכון מחושב: ₪2 מיליארד שנתי בהרחבה לאומית (מודל כלכלי ביטוח לאומי)

### 4. פריצת הדרך (Breakthrough)
**הכותרת:** "ראשון בישראל: AI שרואים איך הוא חושב"

- אלגוריתם ראשון מסוגו — לא ממוצע, אישי ומוסבר
- Agentic Loop מלא: תמלול → ניתוח → התאמה → CRM → Nudge (2.3 שניות, 0 ידנית)
- מודל 1:75 — מלווה חברתית + AI = 75 אזרחים בליווי רציף
- Amazon Bedrock, Transcribe, Personalize, Lambda, S3, DynamoDB

### 5. ישימות (Feasibility)
**הכותרת:** "לא PoC — פיילוט חי"

- 286 אזרחים ממשיים, פסגת זאב ירושלים, 5 שכונות
- 3 מלוות חברתיות, 18 ספקים, 102 שירותים ממופים
- מבוסס החלטות ממשלה 127 ו-150 (רפורמת הסיעוד)
- מוכן להרחבה: 5 ערים ב-2026, קול קורא ממשלתי ב-Q4

## ארכיטקטורת AWS — רשימה מדויקת

### שירותים בשימוש (Production-ready בפיילוט)

| שירות | שימוש ספציפי | מודל עלות |
|-------|-------------|-----------|
| **Amazon Bedrock (Claude Sonnet 3.7)** | ניתוח אינטייק · ייצור המלצות · צ'אט אזרח (לימור) · צ'אט מלווה (CoordinatorAI) · סיכום שיחות | per-token |
| **Amazon Transcribe** | תמלול אינטייק קולי בזמן אמת · זיהוי שפה (עברית/ערבית/רוסית) | per-minute |
| **Amazon Personalize** | אלגוריתם המלצות מותאם אישית · עדכון מודל שבועי לפי engagement | per-event |
| **AWS Lambda** | Service Discovery Agent (שבועי) · Nudge Engine (יומי + event) · Deterioration Monitor · API Gateway backend | per-invocation |
| **Amazon DynamoDB** | פרופילי אזרחים (286 records) · engagement events · alert log | per-RCU/WCU |
| **Amazon S3** | קטלוג שירותים (102 JSON) · הקלטות אינטייק · מסמכים · אסטים סטטיים | per-GB |
| **Amazon API Gateway** | REST API לממשקים · SSE streaming לצ'אט · webhook למידע חוזר | per-call |
| **Amazon SNS / SQS** | Nudge delivery (WhatsApp/SMS) · Agent coordination queue | per-message |
| **Amazon CloudWatch** | ניטור agents · latency · error rate | per-metric |

### ארכיטקטורה — Data Flow

```
[אזרח/מלווה] → API Gateway → Lambda
                               ↓
                        Bedrock (Claude)  ←── context: citizen profile (DynamoDB)
                               ↓                       + service catalog (S3)
                        [analysis JSON]
                               ↓
                     Matching Engine (Lambda)
                               ↓
                     DynamoDB ← recommendations
                               ↓
                     SNS → WhatsApp/SMS Nudge

[אינטייק קולי] → Transcribe → Lambda → Bedrock → DynamoDB (profile update)
                                                         ↓
                                              Personalize (retrain)
```

### עלות פיילוט (הערכה חודשית — 286 אזרחים)
- Bedrock: ~$200/month (ניתוח + צ'אט)
- Transcribe: ~$15/month (50 אינטייקים × 10 דקות)
- Lambda + API GW: ~$5/month
- DynamoDB + S3: ~$10/month
- **סה"כ: ~$230/month** ≈ $0.80/אזרח/חודש vs. ₪8,000 עלות סיעוד ממוצעת

**"האם הנתונים אמיתיים?"**
> 286 אזרחים ממשיים, פסגת זאב, ירושלים. 102 שירותים ממופים בשטח. 
> מלוות חברתיות שכבר עובדות עם המערכת. הנתונים האישיים — אנונימיזציה מלאה.

**"למה AWS ולא מה שיש?"**
> Bedrock מאפשר lLLM בתוך ה-VPC הממשלתי — compliance + data residency.
> Transcribe עם עברית/ערבית מחוץ לקופסה. Personalize מתאמן על engagement בלי data science.

**"מה מנע פתרון כזה עד היום?"**
> שלוש בעיות נפתרו במקביל: (1) LLM שמבין עברית (Bedrock/Claude). 
> (2) Explainability שמאפשר אמון רגולטורי. (3) עלות: $0.80/אדם/חודש, לא $80.

**"מה הצעד הבא?"**
> קול קורא ממשלתי Q4 2026 לחמש ערים. ביטוח לאומי כ-anchor funder. 
> כבר יש תמיכה של יואב בן-דוד ואורי כהן ממשרד האוצר.
