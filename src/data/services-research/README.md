## המטרה
מערכת AI שמזהה, ממפה, מדרגת ומתאימה שירותים לאזרחים ותיקים — באופן אוטומטי ומתעדכן.

## הערך
- **לאזרח:** "מצאנו בשבילך את מה שהכי מתאים — ליד הבית, בשפה שלך, חינם"
- **למלווה:** "לא צריכה לחפש — המערכת ממפה ומדרגת בשבילך"
- **למדינה:** 20% דחיית הדרדרות = ₪2 מיליארד חיסכון שנתי

## Flow — מקצה לקצה

```
[אייג'נט שבועי]
    סורק 47 מקורות → מוצא שירותים חדשים/שינויים
    → מעדכן services-catalog.json
    → מתריע: "🆕 3 שירותים חדשים" / "⚠️ שירות נסגר"
         ↓
[מנוע התאמה — 5 שכבות]
    לוקח פרופיל אזרח (מוטיבציות, חסמים, שכונה, שפה)
    × קטלוג שירותים (102+)
    = ציון התאמה לכל שירות
    → מציג top 5-8 המלצות
         ↓
[אזרח + מלווה]
    בוחרים שירות → מזמינים
         ↓
[אחרי השירות]
    דירוג (1-5 כוכבים) + "היית ממליץ?"
    → חוזר למנוע ההתאמה (שכבה 5: הוכחה חברתית)
         ↓
[לולאת למידה]
    דירוגים משפיעים על המלצות עתידיות
    שירותים מדורגים נמוך → בדיקה → הסרה/שיפור
```
